# 무인 단속 카메라 재배치 지도 시각화


### 시각화 지도 실행
`pip install -r requirements.txt` 의존성 설치후 <br>
루트 디렉터리에서 `main/main.py` 실행

### 컨벤션 및 그라운드 룰
모든 코드는 main 에다가 바로 push 하지 않고, 카카오톡 으로 코드 수정사항을 논의하고 PR-Merge 함. <br>
프로젝트 주제를 늦게 바꾸게 되어 급하게 하다보니 그렇게 규칙을 설정하게됨. <br>
매주,매일 팀원들과 프로젝트 수업으로 만나고, 소통하기 때문에 Git flow 와 PR 메시지 규칙은 굳이 적용하지 않음 <br>
